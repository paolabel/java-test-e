## Teste de projeto java

### Requisitos
---

- Maven
- Java 11

### Execução
---

``` bash
mvn spring-boot:run
```

Aplicação irá rodar em ```http://localhost:9564```

### Execução individual dos testes
---

``` bash
mvn test
```

### Referência da API
---

Disponível em ```http://localhost:9564/```

### Acessar console do banco de dados
---

```http://localhost:9564/h2-console```

senha = password
